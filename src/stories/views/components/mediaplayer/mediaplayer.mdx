# Mediaplayer

## Beschreibung

Die Mediaplayer-Komponente kann, abhängig von der Konfiguration, einen Video- oder einen Audio-Player erzeugen. Technische Basis
dieser Player ist der ARD-Player. Wird sie in einem Teaser verwendet wird ein statisches Teaser-Bild mit einem Play-Button
dem eigentlichen Player vorgeschaltet. Erst bei Klick auf den Play-Button wird der eigenliche Player geladen.
